## 项目简介

LinQ是一个致力于在完全本地化环境下，通过持续对话和数据库记忆实现基于用户偏好记忆的个性化对话以助力生活学习的AI伙伴项目。
现已通过本地部署的Ollama模型，完成了从电脑端到手机端的全栈开发与部署。

## 核心功能

- **持续结构化记忆**：基于SQLite与JSON，构建动态演进的关键词与场景记忆库，使AI伙伴能进行有连续上下文的对话。
- **全栈跨平台部署**：核心逻辑使用Python开发，并成功通过BeeWare框架将应用打包为原生Android APP,实现了从电脑端到移动端的完整迁移。
- **数据同步**：支持通过JSON文件导入/导出数据库，实现数据迁移。
- **具身化演进**：项目蓝图已规划ESP32传感器集成，旨在为AI伙伴赋能物理世界的感知与交互。

## 技术栈

- **AI与后端**: Python，Ollama，SQLite
- **移动端框架**: BeeWare (Android)
- **部署与环境**: Termux (Android)
- **版本控制**: Git
-  **嵌入式（未来）**: ESP32
